This page is going to contain things you need to know for good web design.

First here are the basics:

  • HTML - Hyper Text Markup Language
  • XHTML - Extensible Hyper Text Markup Language
    • Case sensitive - no caps in elements or attributes
    • Attributes must be quoted
    • Requires closing tag
    • Tags must be properly nested
  • CSS- Cascading Style Sheets
    • Separates format from content
    • Gives more control to designers
    • Three ways to include style: inline, internal (document), and external
